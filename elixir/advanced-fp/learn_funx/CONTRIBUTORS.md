## How to Contribute

We welcome contributions that improve the learning experience! See the bottom of [README.md](README.md) for contribution guidelines.

## Types of Contributions Welcomed

- 🐛 Bug fixes in domain logic or tests
- 📚 Improvements to learning materials (`example.txt`, `questions.me`)
- 🧪 Additional test examples that demonstrate functional patterns
- 📝 Documentation clarifications
- 💡 New domain entities or business rules that showcase functional concepts
- 🔧 Better LLM guidance or usage rule improvements

## Recognition

Contributors who make significant improvements to the learning experience will be recognized here.
